# Clinical Reasoning Templates

## Medication Review Template

```
<think>
1. Current medications: [list]
2. New medication proposed: [name, dose, route]
3. Interaction check:
   - Drug-drug: [findings]
   - Drug-disease: [findings]
   - Drug-food: [findings]
4. Contraindications: [list]
5. Dose adjustment needed: [renal/hepatic considerations]
6. Monitoring parameters: [labs, vitals, symptoms]
</think>
```

## Differential Diagnosis Template

```
<think>
1. Presenting complaint: [summary]
2. Key features:
   - Positive findings: [list]
   - Pertinent negatives: [list]
3. Differential diagnoses:
   a. [Most likely] - Supporting: [...] Opposing: [...]
   b. [Second likely] - Supporting: [...] Opposing: [...]
   c. [Cannot miss] - Supporting: [...] Opposing: [...]
4. Recommended workup: [tests to differentiate]
5. Red flags to monitor: [list]
</think>
```

## Literature Review Template

```
<think>
1. Clinical question (PICO format):
   - Population: [who]
   - Intervention: [what]
   - Comparison: [alternative]
   - Outcome: [measure]
2. Search strategy: [terms, databases]
3. Evidence found:
   - Study 1: [design, N, key finding, LOE]
   - Study 2: [design, N, key finding, LOE]
4. Evidence synthesis: [agreement/conflict between sources]
5. Limitations: [gaps, biases, applicability]
6. Conclusion strength: [strong/moderate/weak/insufficient]
</think>
```
